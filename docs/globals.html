TStrategy

TStrategy: "shallow" | "merge" | "deep" | TStrategyFn
  • Defined in types/slim.ts:32

Used for object merging when both a default and a value for an environment exist. It can be passed as a first argument to TDefineFn. It should be either one of the preset strategies ('shallow', 'merge', 'deep') or a TStrategyFn function. See merge-strategies for further details on the preset strategies.
